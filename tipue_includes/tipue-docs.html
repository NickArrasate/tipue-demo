These instructions will help you get Tipue site search integrated into your site.

First, copy the tipuesearch directory into the top level of your project.

You will need to include the tipue_includes directory into your project's includes directory.

Then, you will need to place this line in the head.html include file: {% raw %}{% include tipue_includes/tipue-head.html %}{% endraw %}

And this one in the footer.html include: {% raw %} {% include tipue_includes/tipue-footer.html %} {% endraw %}

Finally, include {% raw %}{% include tipue_includes/tipue-search-box.html %}{% endraw %} wherever you would like the search bar to appear.
